## Installation
To work with this project locally, first install all dependecies:
1. Clone the repo
```sh
git clone https://github.com/fissy/moniee-mobile.git

```
2. Install Yarn packages
```sh
yarn install 
```
3. Start the application 
```sh
yarn start 
```
